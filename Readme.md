# Proyecto Buscador de Música – API Deezer
Frisly Alejandro Melchor

## 📌 Descripción
Este proyecto es un buscador de canciones que utiliza la API pública de Deezer.
El usuario escribe el nombre de una canción o artista y se muestran los resultados en forma de tarjetas.

Cada tarjeta incluye:
- Imagen del álbum
- Nombre de la canción
- Nombre del artista
- Una vista previa de 30 segundos
- Consumo de API pública
- Fetch, async/await
- Creación de elementos dinámicos en HTML
- Manejo de errores y mensajes al usuario

---

- Se espera la respuesta usando `async/await`.
